# Calculadora de Calorias y CrossFit

App web instalable para registrar comidas, macros, entrenamientos, calorias quemadas y balance semanal.

## Funciones

- Resumen diario de calorias consumidas, quemadas y netas.
- Graficos de donut y barras para visualizar el progreso.
- Registro de comidas con entrada rapida y macros.
- Registro de entrenamientos por captura o texto libre.
- Estimacion de calorias quemadas por deporte, duracion, peso y esfuerzo percibido.
- Perfil personal con peso, altura, sexo, grasa corporal, habitos y objetivos.
- PWA preparada para instalar en movil.

## Uso local

Abre `index.html` directamente o arranca un servidor local:

```bash
python3 -m http.server 8000
```

Luego entra en `http://127.0.0.1:8000`.
